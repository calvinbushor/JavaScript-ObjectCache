ObjectCache()

Reading, writing and modifying Objects made awesomer.

What are the properties available to ObjectCache?

  • .set(key, val) // Sets a property to your ObjectCache using a kay value pair
  • .get(key) // Returns the value of a key passed in
  • .remove(key) // Deletes a property within your ObjectCache
  • .dump() // Returns the entire ObjectCache

This is what's currently in my Object Cache which was set in app.js

  • Author: "Calvin"
  • Twitter: "twitter.com/calvinbushor"
  • people: Object

Below is the code that creates your Object Cache and an example of how to set the above objects to it.

            // Create your Object Cache
            var oc = new ObjectCache();

            // Set a couple things directly to the obj using key value pairs
            oc.set('Author', 'Calvin');
            oc.set('Twitter', 'twitter.com/CalvinBushor');

            // Set a dynamic object of people within the Object Cache
            for ( var i=0; i<9; i++ ) {
                oc.set('people,person' + i, 'p'+i);
            }

In the Object Cache, there are three things. A persons name, their twitter handle and an object. If you want to access the value of a property within your object see the example below.

            // oc.get(key) returns a value stored in the key if matched and null if nothing matched

            oc.get('Author'); // Returns 'Calvin'
            oc.get('Twitter'); // Returns 'twitter.com/CalvinBushor'

            oc.get('people'); // returns the entire people Object
            oc.get('people,person4'); // returns the value of the 5th person (person4) which is p4

If you wanted to delete a property of your Object Cache see the below example.

            oc.remove('Author'); // Deletes the key Author and its value Calvin from the Object Cache
